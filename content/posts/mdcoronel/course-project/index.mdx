## Project description

The goal of this project is to evaluate how different decoder-only transformer models interpret and reflect a user's desired emotional tone in response to a prompt on a specific topic, focusing particularly on the intensity of anger. This involves the combined tasks of sentiment analysis and emotion intensity quantification within the context of statistical natural language processing (NLP). Sentiment analysis is a foundational task in NLP, and our project extends this by examining the degree of emotion a model can express, helping us assess nuanced emotional tones, especially anger. Using the `monologg/bert-base-cased-goemotions-original` model from Hugging Face, we aim to score and categorize anger intensity in text responses. Our approach involves generating responses to the same prompt with varying intensities and assessing how well the model captures these desired emotional nuances.

- **Nov 18** - identify models for comparison
- **Nov 22** - create prompts (15; 5 topics, 3 sentiments); have sentiment analyzer ready for use
- **Nov 25** - analyze performance of various models; adjust prompts as needed
- **Nov 29** - report results
