## Description

The author proves that a neural network with only one hidden layer and no constraint on the number of nodes is sufficient to approximate any continuous function. (In other words, such a neural network is a universal approximator.) The author shows that various types of functions, such as continuous sigmoidal functions, non-vanishing integrable functions, and identity functions on n-dimensional rectangles, can serve as activation functions in the neural network. He does not make any claim about the number of nodes necessary to approximate a particular function to a fixed amount of accuracy, but he speculates that the number is "astronomical" for for most problems. The proofs are elegant and use several well-known theorems from functional analysis and uniform algebras.
