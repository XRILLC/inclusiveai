## Project description

This project aims to explore the potential shift in the context of words used in TED talks over time. The primary question to answer is: Did words such as "climate", "society", "technology", "health", or "[laughter](https://www.inc.com/bill-murphy-jr/the-25-most-popular-ted-talks-include-this-1-surprising-word-over-over-reason-why-is-eye-opening.html)" change their contextual relationships over time? If there is enough data, we will also attempt to determine whether the shifts in context correlate with the law of conformity as described by [Hamilton, Leskovec, and Jurafsky](https://arxiv.org/pdf/1605.09096)?

## Estimated timeline

* **Week 1**: Gather and process TED talk transcripts to create corpuses
* **Week 2**: Use sentence transformer to create embeddings
* **Week 3**: Examine contextual shifts (if any) and write up results
