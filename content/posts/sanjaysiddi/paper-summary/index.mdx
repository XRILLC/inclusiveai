## Description

This paper explores using machine learning and natural language processing to automatically map course outcomes to program outcomes using a dataset of 247 course outcomes and 12 program outcomes. The authors tested different text feature extraction methods like TF-IDF and Word2Vec with five machine learning models - SVM, Naive Bayes, Multi-Layer Perceptron, Random Forest, and XGBoost, finding that TF-IDF generally performed better. They also tested advanced deep learning models, such as DistilBERT, RoBERTa, and ELECTRA, and found that RoBERTa achieved the best performance with an F1 score of 85.96%. The study concludes that transformer-based models like RoBERTa are particularly effective for small, domain-specific datasets, and suggests future research could explore applying these models in broader, domain-agnostic contexts or developing new algorithms for limited labeled data.

## Motivation

I chose this paper because it aligns with my current internship work at the Spanish & Portugese department, where I'm exploring how NLP and ML can be used to assess whether a course contributes to degree outcomes based on the student learning outcomes in its syllabus. The paper's focus on mapping course outcomes to program outcomes using text analysis techniques gives me valuable insights for applying similar methods to my own project.
