## Task summary

The task is to determine whether two spans of text were written by the same author. The spans are separated by a "[SNIPPET]" delimiter, and the goal is to classify them as either coming from the same author or different authors. The classification is binary, with the output being:

0: Not the same author

1: Same author

## Exploratory data analysis

I noticed a class imbalance, with 1,245 examples of "different author" (Label = 0) and only 356 examples of "same author" (Label = 1). Luckily there were no missing values in any column.
## Approach

- Data Preprocessing: 

    To ensure the text is clean and usable for the model, I first preprocessed the text. This involved converting the text to lowercase to standardize it and removing unnecessary characters  that do not contribute to the analysis. I also replaced the "[SNIPPET]" delimiter with a space preserved marker to separate the two spans of text. I also removed stopwords which don't add significant meaning to the content for the classification purpose.

- Feature Extraction:

    I used TF-IDF to transform the cleaned text into numerical vectors. This technique helps quantify the importance of words in the context of the entire dataset, allowing the model to focus on significant terms while ignoring less informative ones. I chose to include unigrams, bigrams, and trigrams in the feature extraction process to capture a broader context from the text.

- Training:

    For the classification task, I selected Logistic Regression due to its simplicity and effectiveness in binary classification. To optimize the model, I used Grid Search to tune hyperparameters such as regularization strength and solver type. I evaluated the model using 5-fold cross-validation to ensure it generalizes well on unseen data.
## Results

Once the model was trained and evaluated, I applied it to the test data. The model predicted whether each pair of spans was written by the same author (Label = 1) or different authors (Label = 0). In this case, 112 rows were classified as Label = 1 (same author) and 787 rows as Label = 0 (different authors). I achived an F1 score of 0.4738 in the kaggle competition leaderboard.
