## Exploratory data analysis

_See [the rubric](https://parsertongue.org/courses/snlp-2/assignments/rubrics/class-competition/final/#approach)_

The function calculate_word_frequencies calculates the frequency of each word in texts corresponding to a specific label. This creates new features that idnetify the most common words in the dataset for each class to capture patterns or stylistic differences between the author's and non-author's writing.

extract_text_features removes additional features from each text snippet. The column author_word_count is a count of words from the most common words associated with the author's texts.  non_author_word_count is the count of words from the most common words associated with non-author's texts. The total number of words in the text is also added as text_length. This function converts raw text into numerical features to provide extra context about word usage and text length, which are added to the input for the model.

The model produced an F1 score of 0.59906
