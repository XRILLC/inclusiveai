## Task summary

The task involves predicting whether two spans of text were authored by the same person. This is a binary classification problem, where labels are 1 (same author) and 0 (not the same author). The dataset was provided in CSV format, and the evaluation metric is the Macro F1 score.

## Exploratory data analysis

Initial analysis revealed:

- Text spans vary in length, requiring preprocessing for consistent feature extraction.
- Labels are imbalanced, with more 0s (not the same author) than 1s.
- Texts are rich in stylistic and lexical cues, ideal for feature extraction using TF-IDF.

## Approach

### Data Preparation:

- Split the dataset into training (80%) and validation (20%) sets.
- Text spans were preprocessed using TF-IDF vectorization to represent textual data numerically.
- Limited features to the top 5000 for computational efficiency.
- Model Selection:
  - A Random Forest Classifier was chosen for its robustness and ease of interpretability.
  - The pipeline was built with TfidfVectorizer for feature extraction and RandomForestClassifier for classification.
- Validation:Achieved a Macro F1 score of 0.4338 on the validation set, indicating the need for further tuning.

- Test Performance:
Achieved a Macro F1 score of 0.40638 on the test set, slightly lower than validation performance, indicating potential overfitting or generalization challenges.

### Submission:

Predictions on the test dataset were saved in the required submission format.

##  Results

Validation Macro F1 Score: 0.40638

## Error analysis

- Class Imbalance: The model failed to predict the minority class (1: same author) effectively, as evident from low recall.
- Text Length Variance: Large disparities in span lengths likely impacted feature uniformity.
