## Task summary

The task involves predicting whether two spans of text were authored by the same person. This is a binary classification problem, where labels are 1 (same author) and 0 (not the same author). The dataset was provided in CSV format, and the evaluation metric is the Macro F1 score.

## Exploratory data analysis


- Text spans vary in length, requiring preprocessing for consistent feature extraction.
- Labels are imbalanced, with more 0s (not the same author) than 1s.
- Sentence length distributions for both classes show notable differences.
  - Author 1(1s) generally uses shorter sentences, while other authors(0's) exhibits more variability.
- Word count and character count histograms reveal additional patterns in text length distribution.
- Texts are rich in stylistic and lexical cues, ideal for feature extraction using TF-IDF.

## Approach

### Data Preparation:


- The dataset was split into training (80%) and testing (20%) sets, stratified to preserve class distribution.
- Text spans were preprocessed using TF-IDF vectorization, capturing up to 10,000 features with unigrams and bigrams.
- Added feature enrichment by extracting stylometric features such as:
  - Average word length
  - Average sentence length
  - Lexical diversity
  - Punctuation count
  - Readability score (using textstat).

### Model Selection:
  - A Neural Network Model was used instead of a Random Forest classifier. The model architecture consisted of:
    - Input Layer: Combined TF-IDF and stylometric features.
    - Two Dense Layers with 128 and 64 neurons, ReLU activation, and 30% dropout for regularization.
    - Output Layer: A single neuron with a sigmoid activation for binary classification.
  - The model was compiled with the Adam optimizer and binary cross-entropy loss. 

### Class Imbalance Handling:

- Used class-weight adjustments during training to counter the imbalanced dataset (class_weight='balanced').

### Feature Engineering:

- Combined TF-IDF features with handcrafted stylometric features to enhance the neural network's ability to distinguish authorship.


### Submission:

Predictions on the test dataset were saved in the required submission format.


### Results
- Validation:Achieved a Macro F1 score of 0.51 on the validation set, indicating the need for further tuning.

- Test Performance:
Achieved a Macro F1 score of 0.52325 on the test set, slightly more than validation performance, suggesting improved generalization to unseen data or possible variations due to data distribution.


## Error analysis

- Class Imbalance: The model failed to predict the minority class (1: same author) effectively, as evident from low recall.
- Text Length Variance: Large disparities in span lengths likely impacted feature uniformity.

## Challenges 

In this task, data augmentation poses unique challenges due to the nature of authorship attribution. Techniques like masking, replacing, or deleting words can inadvertently alter the author's writing style, which is central to the task. For instance:

- Synonym Replacement or Word Substitution:
Replacing words with synonyms can disrupt the author's characteristic use of repeated words or unique phrasing patterns, potentially weakening the model's ability to recognize stylistic consistency.

- Word Deletion:
Removing words risks altering the meaning of sentences, which may misrepresent the author’s intended style and nuances.

- Masking Words:
While masking introduces some variation, it does not significantly impact the author's writing style or provide meaningful diversity for training.

Thus, careful consideration is needed to design augmentation techniques that preserve stylistic elements while introducing meaningful variation.
