## Description

This paper is about  using non linear method for text regression problem using convolutional neural network.
The authors used movie reviews and movie attributes to predict the future box-office revenue of the movie.
This paper also compares the results of a traditional linear regression results with the non linear regression model results.

## Motivation

_Why did you select this paper?_
According to the universal approximation theorem, any distribution can be approximated given enough parameters and sample data. 
When train a Neural Network, the time to train and accuracy mainly depend on hyperparameters and training data. 
This method uses a non linear approach, resulting in better performance whith lesser training data. 
The linear model requires huge data where as in the proposed model there is a 40% improvement with the same size of training data.
