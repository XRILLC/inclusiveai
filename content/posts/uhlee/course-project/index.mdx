## Project description

This project aims to compare several open-source generative models' capabilities in text-summarization, using song lyrics from The Beatles. From there, the summaries would be evaluated against human-written summaries. The metrics for evaluation include accuracy on the theme, coherence, and relevance.

Timeline for the project:
Nov 18, have dataset ready, access to multiple open-source gen-ai models
Nov 25, have the models' summaries ready and begin evaluation
Dec 2, analyze results and write the report on the findings
Dec 9, finish and turn in the blog with the findings
