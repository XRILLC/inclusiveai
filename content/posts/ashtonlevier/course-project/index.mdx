## Project description
- The goal of our project is to assess how accurately models can interpret a user's desired tone in response to a prompt on a specific topic, while also investigating sentiment analysis within the context of statistical NLP. Sentiment analysis is a key task in NLP, and our project will involve generating and analyzing sentiment in text. We will prompt multiple models with the same question, asking for responses that vary in emotional intensity, and evaluate how well the models capture and reflect the desired tone.

# Project
- Nov 18 - identify models for comparison
- Nov 22 - create prompts (15; 5 topics, 3 sentiments)
- Nov 25 - analyze performance
- Nov 29 - report results
