## Task summary


-The task is to determine if two spans of text were written by the same author. It preprocesses the text, extracts features using TF-IDF, and incorporates differences, similarities, and cosine similarity into the feature set. The model combines Gradient Boosting, Random Forest, and Logistic Regression using a Voting Classifier to predict author similarity.-


## Exploratory data analysis

- 1. Data Inspection
   - Started by loading the `train.csv` and `test.csv` datasets.
   - Checked that the `TEXT` column contained two spans of text separated by `[SNIPPET]`.
   - Verified that the training data had a `LABEL` column with binary values indicating whether the spans are from the same author or not.

2. Splitting Text
   - Split the `TEXT` column into two separate columns, `span1` and `span2`, to make it easier to analyze and compare each span individually.

3. Text Analysis
   - Looked at the word and character counts of `span1` and `span2` to ensure the spans were formatted consistently.
   - Noticed some differences in content and vocabulary between the two spans, providing clues about their potential relationships.

4. Data Cleaning
   - Standardized the text by converting everything to lowercase.
   - Removed special characters (like punctuation) to make the data cleaner and easier to process.

5. Target Analysis
   - Examined the `LABEL` column in the training data to check if the classes (same author vs. different authors) were evenly distributed or imbalanced.

6. Feature Exploration
   - Used TF-IDF vectors to represent the text spans and studied how well these features captured patterns in the data.
   - Looked at cosine similarity scores between spans to understand how similar they were and whether this metric could help distinguish between same and different authors. -
