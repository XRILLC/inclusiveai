## Task summary

I am going to utilize words as vectors to classify the documents. My motivation/general thought process is that authors often use similar groupings of words
which is apart of their writing style. Ideally these vectors partnered with logistic regression will accurately calculate the probability between authors.
I will be reviewing the first set of results and modifying some of the settings for the logistic regression model I'm using to determine if this is a feasible
route to continue on. If not, I will be exploring if there is either a different probability model that would be better suited instead of logistic regression or changing what I use as the features.
