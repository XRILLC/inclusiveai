## Project description

This project aims to explore cross-lingual word embedding matching to create synthetic pairs of phrases between a low-resource language and English. By mapping words and simple phrases from a low-resource language onto English using pre-trained embeddings, the goal is to augment data in cross-lingual NLP tasks.
