## Task summary

My approach involved preprocessing the training data by tokenizing, removing stopwords, and lemmatizing the text. I then employed an improved TF-IDF vectorizer, which includes unigrams and bigrams, adjusts for document frequency, and limits the number of features to enhance model performance. The model was trained using logistic regression with class balancing to handle any class imbalance. I validated the model’s performance using the competition's test data and made predictions, which were submitted to the leaderboard for evaluation. Throughout the process, I ensured that the code was well-organized and ready for integration into a version-controlled GitHub repository for collaboration and reproducibility.
