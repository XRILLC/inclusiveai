## Project description

In this project, the goal is to utilize training data to create a machine that can accurately label testing data based on sentiment analysis using a model called BERT, created by Google. The goal is to return a csv file with the labels and turn it in for evaluation. 

Step 1: Choose a project and evaluate the data (COMPLETE)
Step 1.5: Familiarize myself with BERT (Due 11/17)
Step 2: Begin writing code for the project, disregarding proper labelling (Due 11/22)
Step 3: Incorporate the machine learning aspect/labelling (Due 11/29)
Step 4: Fine tune until the score is as high as it can go (Due 12/04)
Step 5: Submit score and fill out index file with relevant information (Due 12/07)
