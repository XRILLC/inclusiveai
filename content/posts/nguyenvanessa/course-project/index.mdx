## Project description

The goal of our project is to assess how different decoder-only transformer models interpret a user's desired emotional tone in response to a prompt on a specific topic, including when the prompt requires varying levels of intensity. This involves both sentiment analysis and emotion intensity quantification within the context of statistical NLP. Using the monologg/bert-base-cased-goemotions-original model from Hugging Face, we will score and categorize the intensity of a requested emotion in text responses. We will prompt multiple models with the same question, ask for responses that vary in emotional intensity, and evaluate how well the models convey not only the requested emotion but also the specified intensity of that emotion.

# Project
- Nov 18 - identify models for comparison
- Nov 22 - create prompts (15; 5 topics, 3 sentiments); ready sentiment analyzer for use
- Nov 25 - analyze performance; adjust prompts
- Nov 29 - report results
