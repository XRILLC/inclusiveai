## Project description

This project aims to explore how different LLMs react to user inputs and to see if an LLM is able to predict the preferred responses of users. To build the dataset, users chatted with two anonymous LLMs and selected which answer they preferred from the two, creating a dataset of user prompts, LLM responses, and user judgements. The project is intended to demonstrate how human feedback can assist with creating better models, leading to a better human experience. The goal of the competition is to build a model that can accurately predict the preferences of the judges, determining the likelihood that a test prompt/response pair is the ideal choice. The accuracy of submissions is based on the log loss between the predicted probabilites and the ground truth values.
