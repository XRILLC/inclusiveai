## Task summary

The task that we have been given is to take two sets of text and use a classification model to determine which sets were produced by the same author. I plan to make a first pass using a similar approach to one in LING539, which was to use pandas to view and find data outliers on the .csvs given and then use CountVectorizer to creature features, LogisticRegression to train the model on those features, and then create predictions for the test.csv and output the results. I'd like to see how the model performs when building its own features with scikit-learn, and then move on to a new approach such as working with a BERT classifier, as I haven't tried that before.
